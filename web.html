Game Sniper

Overview

Developed a full-stack web application that allows users to search and browse discounted video game deals using external gaming deals APIs. Built with Node.js, Express.js, MongoDB, and EJS, the application supports user registration/login functionality and dynamic rendering of deal information retrieved from third-party APIs. The project was deployed online using Render and designed as a complete end-to-end web application integrating backend routing, database management, API consumption, and responsive frontend rendering.



Features Implemented

  • Built a Node.js and Express.js backend with multiple GET and POST routes for authentication, registration, and game deal retrieval
  • Implemented user registration and login functionality using MongoDB and Mongoose for persistent user data storage
  • Created dynamic EJS templates to render user profiles, login pages, registration forms, and API-driven game deal results
  • Connected to an external video game deals API to fetch real-time discounted game pricing and deal information
  • Processed and rendered API response data dynamically, including game titles, thumbnails, sale prices, and original prices
  • Designed MongoDB schemas and database helper functions for storing and retrieving user account information
  • Configured static asset serving, environment variable management with dotenv, and deployment support for cloud hosting platforms

SuperMarket Web Application

Overview

Developed a full-stack supermarket ordering web application using Node.js, Express.js, and EJS templating that allowed users to browse available grocery items and place online orders. The application loaded product information dynamically from JSON files, generated catalog and order pages on the server side, and processed customer purchases with automatic cost calculation and order summaries. The project also included a command-line server interface for managing server operations and displaying inventory data directly from the terminal.



  • Built an Express.js web server with multiple GET and POST routes for catalog browsing, order placement, and order confirmation
  • Developed a command-line interpreter that handled custom server commands such as displaying inventory data and safely shutting down the server
  • Created a reusable ES6 class with private instance variables for reading and managing supermarket inventory data from JSON files
  • Parsed and processed JSON product data dynamically using Node.js file system operations
  • Generated dynamic HTML pages using EJS templating for catalog displays, order forms, and order confirmation summaries
  • Implemented order processing logic that calculated total purchase cost based on selected grocery items
  • Added support for both single-item and multi-item purchases using array-based request handling
  • Dynamically generated HTML tables and dropdown menus from inventory data without hardcoding product information
  • Applied functional JavaScript array methods (forEach, find) instead of traditional iteration statements for array processing


  • Node.js
  • Express.js
  • EJS Templating
  • JavaScript ES6 Classes
  • JSON Data Processing
  • Command-Line Interfaces
  • HTML/CSS
  • Functional Array Methods
  • File System Operations
